Project: Restaurant Menu APP
This is a course project for Udacity FullStack Web Developer Nano Degree: Item Catalog.

Users can login via Google+ and once verified they can add/edit/delete their own restaurants/menu items.

Which also provides the JSON API endpoints:
/restaurant/<int:restaurant_id>/JSON   - will give items per restaurant
/restaurant/JSON - will give restaurants 


Setup:
1. run database.py to create a database
2. run initial_the_database.py to add restaurants and menu items to the DB.
3. run project.py

visit http://localhost:5000/ to view the app.
